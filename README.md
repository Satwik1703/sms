# SMS
Architecture Assignment 4
A microservice designed mimic the facilities of SMS
Built using
	- Node
	- Express
	- MongoDB

## Instructions to setup
```bash
$ git clone https://github.com/Satwik1703/sms.git

$ cd sms

$ npm install
```
##Running the code
```bash
$ node server.js
```

## Testing the APIs
```bash
$ npm run test
```
